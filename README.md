# 📊 專案名稱：Power BI 數據分析報表

這是一個數據分析專案，透過 Power BI 呈現關鍵指標。請依照以下步驟完成環境設定與報表載入。

--- 


## 📥 1. 下載 PBIT 檔案
請從本儲存庫下載 Power BI 範本檔案（.pbit），這是報表的核心結構。

* **檔案名稱：** `我的作品.pbit`
* **下載方式：** 點選檔案後，點擊右側的 **Download** 按鈕。

> [!TIP]
> PBIT 檔案僅包含報表版面與公式，不包含敏感資料，因此檔案體積較小。

---

## 📂 2. 下載 .csv 原始資料
報表需要讀取底層數據才能運作。

1.  進入本專案的 `/data` 資料夾。
2.  下載所有的 `.csv` 檔案。
3.  **建議：** 將 `.pbit` 與所有 `.csv` 檔案存放在同一個資料夾內，避免路徑出錯。

---

## 🚀 3. 使用方法

### 步驟一：開啟範本
按兩下開啟 **「我的作品.pbit」**。系統會自動啟動 Power BI Desktop。

### 步驟二：連結資料來源
開啟後，系統通常會彈出對話框要求您輸入「參數」或「資料路徑」：
* 請將您的電腦中存放 `.csv` 的 **資料夾路徑** 貼上。
* 點擊 **載入 (Load)**。

![範例圖示：如何輸入資料路徑](https://via.placeholder.com/800x400?text=Power+BI+Loading+Data+Path+Screenshot)

### 步驟三：更新資料 (選用)
如果圖表顯示錯誤，請點選上方工具列的 **「轉換資料」 > 「資料來源設定」**，重新導向正確的檔案路徑。

---

## 🛠 系統需求
* **軟體：** Microsoft Power BI Desktop (建議更新至最新版本)
* **作業系統：** Windows 10 / 11
